# Backend Solution

1. `cd` into this folder
2. Run the `docker compose up -d` command  
   It should run all the services.

After, you should be able to access the backend here: `http://localhost:5000/api/v1`.  
You can test this by visiting the health endpoint: [http://localhost:5000/api/v1/health](http://localhost:5000/api/v1/health)

You are also provided with a PHPMyAdmin to view the database with ease. It's available here: [http://localhost:8888](http://localhost:8888)
